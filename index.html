Less 、Sass 、Scss 、 Stylus
简介 + 区别

CSS预处理器

  1. Less Stylus基于JavaScript, 在客户端处理,Sass基于Ruby,在服务器端处理
  2. 扩展名:Less style.less , Sass style.sass/scss , Stylus style.styl
  3. 语法：变量、Mixins (混入)、继承...

安装 + 配置 + 解析...

  1. LESS只需引入JS文件
    PS：在引入“.less”文件中，“link”的“rel”属性要设置为“stylesheet/less”。LESS源文件一定要在“less.js”引入之前引入，这样才能保证LESS源文件正确编译解析，当然也可在服务器端安装，基于Node JS环境安装, 安装方法就不细讲。。。
  2. Stylus和Less 服务端的安装方法类似，同样依赖于Node JS的环境，唯一不同的是在Node JS的Command控制面板输入的命令不一样：
    $ npm install stylus

SASS 安装方法

  1. 到RubyInstaller官网上下载Ruby安装文件
  3. 输入以下命令:gem install sass
  4. 接下来就可以通过以下命令对style.sass进行时时编译，先找到自己项目的sass根目录，然后：sass -watch style.scss:style.css

compass+sass写css

安装命令:gem install compass

Compass 实用功能

  • compass watch 实时编译scss
  • 通过配置config.rb实现css自动压缩：
  • 图片自动合成雪碧图，并且生成坐标
  • 取img 宽高
  • base64转码

配置config.rb实现css自动压缩

output_style = :expanded(带注释)，compressed(不带注释)

图片自动合成雪碧图，并且生成坐标

@import "../images/*.png"; 图片路径
$images-layout:vertical; 图片合并风格(vertical垂直 horizontal水平 diagonal对角线 smart智能)
@include all-images-sprites;

取img 宽高

.icon-w-h{ width:image-width('../images/icon2.png'); height:image-height('../images/icon2.png'); }
生成后的代码：
.icon-w-h { width: 64px; height: 16px; }
